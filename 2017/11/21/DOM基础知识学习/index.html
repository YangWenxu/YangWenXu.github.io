DOM基础知识学习

刚刚学习了DOM的相关操作，可能是太笨了吧，感觉还是很有难度很抽象的，所以就想查找一些资料总结记录一下这些知识点。DOM是针对xml经过扩展用于html的应用程序编程接口，我们又叫API。DOM给我们提供了用程序来动态控制html的接口，因此，DOM处在javascript赋予html具备动态交互和效果的能力的核心地位上。

DOM结点：
DOM使用对象的表示方式来表示对应文档的结构和属性
DOM结点分为元素节点，文本节点，注释节点等

获取DOM节点的几种方法：

方式一 ：通过getElementById获取节点，得到的是唯一的一个标签对象，因为文档中的ID是唯一的
方式二 ：通过getElementsByTagName获取节点。获取指定标签名的元素或元素集合或相同类的集合。
         通过getElementsName获取元素。获取的是元素名称或者名字相同的一个集合。
         document.getElementByTagName("p")可以得到文档中所有的p标签
         document.getElementByTagName("p")[i]可以逐一访问获取到的P标签
方式三： 通过getElementByClassName获取节点，获取文档中指定class名的标签
方式四： 通过querySelector/querySelectorAll获取节点，获取到的是符合传入css选择器的第一个元素或者全部元素

DOM节点遍历：

如果要获取一个唯一的节点，可以通过节点间的直接关系，来遍历DOM树。
var tree = document.getElementById('tree');
tree.parentNode; //获取父节点
tree.firstChild/tree.lastChild;  //获取第一个或者最后一个子节点
tree.child Nodes/tree.children; //获取所有的子节点、所有的子(元素节点)
tree.previousSibling/tree.nextSibling; //获取前一个/后一个兄弟节点
tree.previousElement Sibling/tree.nextElementSibling; //获取前一个/后一个兄弟元素节点

创建DOM节点：

tree = document.createElement(tagname);  //创建指定标签名称的节点

修改DOM节点：

textContent获取节点及其后代节点的文本内容
tree.textContent;  //一锅
tree.textContent="大锅";  //将一锅修改为大锅
innerText获取节点及其后代节点的文本内容，使用方法与textContent一样，但是兼容的浏览器不一致

插入DOM节点:

appendChild方法在指定元素后追加一个元素节点
var h1 = document.createElement(h1);
h1.innerText="一锅";
tree.appendChild(h1); //在ID为tree的标签中插入一个h1标签
insertBefore方法在指定的节点前面插入一个节点。
tree.insertBefore(ul,h1); //在id为tree的标签中，h1前面插入一个ul标签

删除DOM节点:

removeChild方法删除节点;
tree.removeChild(h1); //删除h1节点，等价于h1.parentNode.removeChild(h1);

innerHTMl: innerHTMl表示节点的HTML内容，可以批量创建节点，插入节点，删除节点
