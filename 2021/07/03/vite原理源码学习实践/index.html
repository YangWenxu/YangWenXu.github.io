下一代构建工具： vite

vite原理

特性

  1. 去掉打包，使用ESbuild进行预构建，使项目的启动可以秒开
  2. 真正意义上的按需加载，热更新
    Vite 提供了一套原生 ESM 的 HMR API。 具有 HMR 功能的框架可以利用该 API 提供即时、准确的更新，而无需重新加载页面或清除应用程序状态
